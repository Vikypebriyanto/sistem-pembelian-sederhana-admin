- Buka: 👉 https://ollama.com/download

- Klik tombol Download for Windows.

- File .msi atau .exe akan ter-download.

- Jalankan file installer — Next → Next → Finish.

Cara menjalankan:

```bash
npm init -y
npm install express mysql2 ejs body-parser
